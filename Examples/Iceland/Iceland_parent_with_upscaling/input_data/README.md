ROMS will look in this directory for input data such as the model grid and forcing files.
These data can be generated by following the 3rd step of the nested CDR tutorial at
`$ROMS_ROOT/docs/tutorials/nested_cdr_tutorial/3_parent_simulation_re-run_with_upscaling.ipynb`.
Note that in order to produce inputs for this configuration, the previous two simulations at
`../Iceland_parent` and `../Iceland_child` will have to be run first.

Guidance on how to run the prerequisite simulations can be found by following the tutorial in full,
including the previous two steps at
`$ROMS_ROOT/docs/tutorials/nested_cdr_tutorial/1_parent_simulation.ipynb`
`$ROMS_ROOT/docs/tutorials/nested_cdr_tutorial/2_child_simulation_with_cdr.ipynb`
