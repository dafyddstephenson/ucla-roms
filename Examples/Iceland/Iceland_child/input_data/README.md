ROMS will look in this directory for input data such as the model grid and forcing files.
These data can be generated by following the tutorial notebook at
`$ROMS_ROOT/docs/tutorials/nested_cdr_tutorial/2_child_simulation_with_cdr.ipynb`
Note that some of the input files for this simulation will have to be converted from outputs
produced by the parent simulation at `../Iceland_parent/`, which will have to be run first.
Guidance on running the parent simulation can be found at `$ROMS_ROOT/docs/tutorials/nested_cdr_tutorial/1_parent_simulation.ipynb`
